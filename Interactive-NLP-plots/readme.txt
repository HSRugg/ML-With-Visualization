These plots show parts of speach counts, important TFIDF n_grams, and sentamint.
From an avalbe 40000+ books thanks to project gutenberg. 


TO USE THE PLOTS:

Download all files to the same folder.
Open treminal, go the folder fith the plots and pikled data. 

Run the comand:

bokeh serve --show FILE_NAME.py


You will need the following python modules:

      pandas, numpy, scipy, bokeh, sklearn, pickle, nltk, gutenberg
      
      For help with the gutenberg module go to the link below.
      https://pypi.python.org/pypi/Gutenberg
